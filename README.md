# -v1.0
本系統是基於Windows操作系統的JDK1.8開發環境，通過eclipse，HBuilder，Navicat開發工具，使用Spring Boot（Spring Boot + Mybatis）框架，B / S模式，實現了一個智能排課系統，用於解決將龐大的課時，眾多的教師，複雜的教室信息合理，迅速，高效地制定相應的排課計劃，在便捷師生的同時，降低資源衝突風險，進行教學管理的數字化和信息化。
